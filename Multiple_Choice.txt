Multiple Choice (20 Questions)

1. What is the correct syntax to declare a function in JavaScript?

a) function myFunction {}
b) function myFunction() {}
c) function: myFunction() {}
d) myFunction = function {}
Answer: B

2. Which of the following array methods is used to add elements to the beginning of an array?

a) push()
b) pop()
c) shift()
d) unshift()
Answer: D

3. How do you access the value "John" from the following object?
let person = { name: "John", age: 30 };

a) person.name
b) person["John"]
c) person["age"]
d) person.name("John")
Answer: A

4. Which loop is best suited for iterating over all elements in an array?

a) for loop
b) for...in loop
c) while loop
d) for...of loop
Answer: A

5. What will the following code output?
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[1]);

a) "apple"
b) "banana"
c) "cherry"
d) undefined
Answer: B

6. Which of the following is a feature of template literals?

a) Can only be used with single quotes
b) Allow multi-line strings without escape characters
c) Do not allow variable interpolation
d) Are limited to only one line
Answer: B

7. Which of the following methods returns the keys of an object as an array?

a) Object.values()
b) Object.entries()
c) Object.keys()
d) Object.getKeys()
Answer: C

8. In JavaScript, what does the push() method do?

a) Removes the first element of an array
b) Adds one or more elements to the end of an array
c) Removes the last element of an array
d) Adds one or more elements to the beginning of an array
